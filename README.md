# 🎁 Gift Exchange Bot

Telegram-бот для безопасного обмена NFT-подарками между пользователями.

---

## ⚙️ Установка

### 1. Клонируй репозиторий
```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой переменные окружения
```bash
cp .env.example .env
```
Открой `.env` и заполни:
```
BOT_TOKEN=токен_от_BotFather
ADMIN_ID=твой_telegram_id
```

### 4. Запусти бота
```bash
python bot.py
```

---

## 🚀 Функционал

- 📝 **Создание сделок** — пошаговое создание обмена между двумя участниками
- 🔗 **Deep link** — уникальная ссылка для второго участника сделки
- ✅ **Принятие сделок** — второй участник принимает сделку по ссылке
- 🏆 **Топ-15 обменов** — таблица лучших сделок
- 📞 **Техподдержка** — контакты менеджера и поддержки

### 👨‍💼 Админ-панель (`/admin`)
- 📊 Статистика (сделки, пользователи, баны)
- 📢 Рассылка всем пользователям
- 🚫 Бан / ✅ Разбан пользователей
- 📝 Редактирование баннера приветствия
- 📋 Просмотр последних сделок
- 🔄 Обновление топа обменов

---

## 🔐 Переменные окружения

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) |
| `ADMIN_ID` | Telegram ID администратора |

## ⚠️ Важно

- Добавь `.env` в `.gitignore`, чтобы не публиковать токен
- Бот хранит данные **в памяти** — при перезапуске сделки сбрасываются. Для постоянного хранения подключи базу данных (SQLite / PostgreSQL)
